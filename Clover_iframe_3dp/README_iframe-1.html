Requirements

These are the setup requirements to use the Clover iframe example provided in this directory.

  • Node JS installed to run the sample app
  • A template engine like Pug, Mustache, or EJS (as used in the tutorial) installed to allow for app routing while testing
  • A local server for Prod development

    • Note: A local server is required for this sample because Clover ecomm APIs will not run in a browser for security reasons.

    For detailed instructions on how to use this Clover-hosted iframe sample, refer to: Use the Clover-hosted iframe.
